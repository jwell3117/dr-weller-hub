Dr. Weller's Chart Builder v0.30 Clinic Release

Use locally:
1. Open index.html in Chrome, Edge, or Safari.
2. No internet connection or server is required.
3. The app does not store patient data.

Deploy:
Upload the contents of this folder to the repository root. The main web page must remain named index.html.

Important:
Review generated clinical documentation before placing it in the medical record. The builder assists documentation but does not replace clinical judgment, coding verification, or EHR review.
